# 제품 기획서

## 1. 제품명

`Study Helper for SA`

참고:

- 프로젝트 폴더명은 `Study Helper for SA Tablet`
- 화면 표시명은 `Study Helper for SA`

## 2. 목표 사용자

- 1차 사용자: 초등학교 2학년 아이
- 보조 사용자: 부모

## 3. 해결하려는 문제

- 아이가 영어 문제를 보면 바로 막힘
- 부모가 매번 해석과 힌트를 대신 설명해야 함
- ChatGPT를 직접 쓰면 답을 바로 받아버릴 수 있음
- 갤럭시 태블릿에서 바로 쓸 수 있는 구조가 필요함

## 4. 제품 원칙

- 정답을 첫 문장에 말하지 않는다.
- 힌트 3단계가 먼저 나온다.
- 초2가 이해할 수 있는 쉬운 한국어를 쓴다.
- 사진 한 장 안에서 특정 번호 문제를 고를 수 있어야 한다.
- 비용이 거의 없는 방식으로 먼저 운영한다.

## 5. MVP 범위

### 필수

- 카메라 또는 사진 업로드
- OCR로 문제 읽기
- 문제 번호 인식
- 힌트 3단계 제공
- 마지막 정답 공개
- 음성 입력
- 힌트 카드와 마지막 단계 카드 안 음성 읽기
- ChatGPT 질문 생성
- ChatGPT 공유하기
- 최근 코칭 기록 저장과 다시 열기

### 제외

- 로그인
- 부모용 학습 기록 저장
- 부모 통계
- 정답률 리포트
- 서버 회원 관리

## 6. 주요 사용 시나리오

1. 아이가 문제집을 태블릿 카메라로 비춘다.
2. `2번 문제 도와줘` 라고 말한다.
3. 앱이 문제 문장을 읽고 2번 문제를 중심으로 힌트를 만든다.
4. 아이가 힌트를 듣고 스스로 답을 생각한다.
5. 더 자세한 풀이가 필요하면 ChatGPT 심화 질문을 복사해 추가 질문한다.

## 7. 성공 기준

- 태블릿에서 큰 오류 없이 실행됨
- 카메라/사진/음성 흐름이 끊기지 않음
- 짧은 영어 문제에서 기본적인 OCR과 문제 번호 분리가 됨
- 아이가 정답보다 힌트 중심으로 사용하게 됨

## 8. 향후 확장

- HTTPS 배포
- 실제 태블릿 실사용 테스트
- 문제집 유형별 OCR 튜닝
- ChatGPT API 또는 다른 AI 연결 검토
- 기록 저장 및 부모 화면 추가
