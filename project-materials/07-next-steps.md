# 다음 작업 우선순위

## 1. 현재 상태

지금 태블릿 프로젝트는
`실행 가능한 1차 버전`
입니다.

즉, 아래는 이미 있습니다.

- 태블릿 UI
- 카메라/사진 흐름
- OCR
- 문제 번호 선택
- 힌트 3단계
- 정답 공개
- 음성 입력
- 음성 읽기
- ChatGPT 질문 복사/공유
- 최근 코칭 기록 저장/다시 열기
- PWA 구조
- 미리보기 실행 스크립트
- 문서 폴더

## 2. 다음으로 중요한 작업

### 1순위. 실제 태블릿 테스트

이유:

- 지금은 구조가 맞는지 확인한 단계
- 실제 갤럭시 태블릿에서 카메라, OCR, 음성 품질을 봐야 함

### 2순위. HTTPS 배포

이유:

- 카메라와 PWA 설치를 안정적으로 쓰려면 사실상 필요함

### 3순위. OCR 품질 튜닝

이유:

- 실제 문제집 사진에서 성능 차이가 가장 크게 나는 부분

### 4순위. ChatGPT 연동 강화

현재는 질문 복사 방식입니다.

나중에는:

- 더 좋은 프롬프트
- 문제 유형별 질문 템플릿
- 부모용 복사 템플릿

으로 보강할 수 있습니다.

### 5순위. 기록 확장

- 최근 기록을 부모용 요약으로 내보내기
- 자주 막힌 문제 유형을 별도 묶음으로 보기
- 다시 풀 문제 표시 기능 추가
