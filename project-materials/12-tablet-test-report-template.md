# 갤럭시 태블릿 테스트 기록 템플릿

이 문서는 `Study Helper for SA Tablet` 을
실제 갤럭시 태블릿에서 테스트할 때
한 번의 테스트 결과를 빠르게 적어두기 위한 기록 양식입니다.

복잡하게 길게 쓰기보다
`어떤 환경에서`, `무엇을 눌렀고`, `어디가 잘됐는지/이상했는지`
세 가지만 남기는 데 목적이 있습니다.

## 사용 방법

1. 이 파일을 복사해서 날짜를 붙여 새 메모를 만듭니다.
2. 한 번의 테스트 세션이 끝날 때 아래 항목만 채웁니다.
3. 문제가 있었던 케이스는 `재현 순서` 를 짧게 남깁니다.
4. 다음 개선 작업은 맨 아래 `다음 액션` 에 1줄로 정리합니다.

예:

`2026-04-01-tablet-test-notes.md`

## 빠른 통과 체크리스트

아래는 `통과 / 아쉬움 / 실패 / 미확인` 중 하나로 적으면 됩니다.

- 첫 화면이 정상적으로 열림:
- 상단 제목이 `Study Helper for SA` 로 보임:
- `For Tablet` 이 작은 보조 문구로 보임:
- `사진 찍기` 버튼으로 전체화면 촬영 열림:
- 촬영 후 메인 화면에 결과 미리보기 남음:
- `다시 찍기` 흐름이 자연스러움:
- 사진 업로드로 이미지 불러오기 가능:
- `OCR 전체 읽기` 결과가 보임:
- 문제 번호 칩이 생성됨:
- `문제 도와줘` 입력 후 힌트 흐름이 열림:
- 힌트 1, 2, 3 순서가 자연스러움:
- 정답이 마지막 단계 전에는 숨겨짐:
- `음성 입력` 버튼 동작:
- 듣는 중 / 정리 중 상태 표시 확인:
- 말한 문제 번호가 입력칸에 반영됨:
- 힌트 카드 안 `자동 읽기` 동작:
- 마지막 단계 카드 안 `자동 읽기` 동작:
- `ChatGPT 복사` 동작:
- `공유하기` 동작:
- 최근 코칭 기록 저장:
- 기록 다시 열기:
- 기록 지우기:
