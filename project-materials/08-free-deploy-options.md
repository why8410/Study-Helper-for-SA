# 무료 배포 옵션 메모

## 1. 왜 필요한가

이 태블릿용 앱은

- 카메라
- PWA 설치
- 서비스 워커

를 안정적으로 쓰려면 `HTTPS 주소`가 거의 필요합니다.

그래서 실제 태블릿 사용 단계에서는
무료 정적 배포가 가장 현실적인 선택입니다.

## 2. 추천 후보

### Netlify

- 가장 쉬운 편
- 정적 폴더 그대로 올리기 좋음
- HTTPS 자동
- 현재 프로젝트에 `netlify.toml` 도 추가해둠

### Cloudflare Pages

- 정적 사이트에 잘 맞음
- HTTPS 자동
- 속도도 안정적인 편

### GitHub Pages

- 코드 관리와 함께 가기 좋음
- 다만 Git 설정에 익숙해야 조금 편함

## 3. 현재 프로젝트 기준 추천

가장 추천:

- `Netlify`

이유:

- 현재 폴더 구조가 정적 사이트라 바로 맞음
- 별도 빌드 도구가 필요 없음
- 설정도 단순함

## 4. 최실장님 기준 실제 흐름

1. 태블릿용 폴더를 정리한다
2. Netlify에 업로드한다
3. HTTPS 주소를 받는다
4. 갤럭시 태블릿에서 연다
5. 홈 화면에 추가한다

## 4-1. 비용과 안전 관련 메모

공식 기준으로 `Netlify Free` 는 무료 플랜입니다.

쉽게 말하면:

- 무료로 시작 가능
- 정적 사이트 테스트 용도로 충분
- 무료 플랜은 hard limit 구조라서 한도 초과 시 자동 과금 대신 사이트가 일시 중지될 수 있음

추가로 기억할 점:

- 유료 플랜의 auto recharge 는 기본적으로 꺼져 있음
- 계정 보안을 위해 2FA 설정 가능
- 최실장님처럼 테스트용 링크가 필요한 경우엔 무료 플랜부터 시작하는 것이 가장 자연스러움

실제 사용 기준에서도
가족 단위, 태블릿 1대 중심의 작은 사용량이면
무료 플랜으로 먼저 시작해 볼 만한 구조입니다.
