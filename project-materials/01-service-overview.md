# 서비스 개요

## 한 줄 설명

화면에 보이는 서비스명은 `Study Helper for SA` 이고,
작은 보조 문구로만 `For Tablet` 이 붙습니다.

`Study Helper for SA` 는 갤럭시 태블릿에서 영어 문제를 사진으로 읽고,
정답보다 먼저 힌트를 주는 초등 2학년용 학습 도우미입니다.

## 왜 이 버전이 필요한가

기존 `Study Helper for SA` 는 Mac 로컬 실행 중심이었습니다.

하지만 최실장님 상황에서는:

- 집에 Mac이 없음
- 갤럭시 태블릿에서 사용하고 싶음
- ChatGPT 계정은 이미 있음

이 조건이 있기 때문에
`태블릿에서 바로 열 수 있는 별도 버전` 이 필요했습니다.

## 핵심 목표

- Android 태블릿에서 사용 가능해야 함
- 정답 직답보다 힌트 코칭이 우선이어야 함
- 비용이 거의 들지 않아야 함
- 필요할 때만 ChatGPT를 보조로 이어 붙일 수 있어야 함

## 현재 구현된 내용

- 태블릿용 단일 화면 UI
- `사진 찍기` 버튼으로 여는 전체화면 촬영
- 사진 업로드
- 음성 입력
- 브라우저 내장 음성 읽기
- 음성 입력 상태 버튼 표시
- 클라이언트 OCR
- OCR 전처리로 사진 대비 보정
- OCR 약할 때 2차 보정 읽기
- 문제 번호 인식
- 문제 번호별 힌트 코칭
- 마지막 단계에서 정답과 풀이 열기
- 힌트 카드와 마지막 단계 카드 안 `자동 읽기`
- ChatGPT 복사
- ChatGPT 공유
- 최근 코칭 기록 저장과 다시 불러오기
- PWA 설치 구조

## 현재 한계

- OCR 품질은 사진 품질 영향을 크게 받음
- 아주 복잡한 문제집 페이지는 흔들릴 수 있음
- OpenAI API를 직접 붙인 수준의 풀이 품질은 아직 아님
- 실제 태블릿 사용을 위해서는 HTTPS 환경이 가장 좋음

## 현재 판단

지금 단계는
`태블릿에서 쓸 수 있는 방향으로 잘 출발한 1차 버전`
입니다.

즉,

- 구조는 맞게 잡힘
- 사용 흐름도 태블릿 기준으로 재정리됨
- 다음 단계는 실제 태블릿 테스트와 배포 보강

## 실제 사용 기준 체크

최실장님처럼
`집에서 실제로 갤럭시 태블릿으로 쓸 예정`
이라면 현재 구조는 충분히 현실적입니다.

다만 아래는 기억해야 합니다.

- 링크를 계속 쓰려면 정적 호스팅이 필요함
- 첫 실행은 인터넷 연결이 있는 편이 좋음
- 기록은 태블릿 브라우저 안에만 남음
- 기본 코칭은 ChatGPT 로그인 없이 사용 가능
- ChatGPT 앱으로 더 깊게 이어갈 때만 ChatGPT 로그인 여부가 필요
