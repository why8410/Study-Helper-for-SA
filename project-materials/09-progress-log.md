- ChatGPT용 질문 복사 기능 추가
- PWA용 manifest, service worker 추가

### 실행 보조

- `Start-Study-Helper-for-SA-Tablet.command`

### 사용성 보강

- `공유하기` 버튼 추가
- 최근 코칭 기록 저장과 다시 불러오기 추가
- 최근 기록 직접 지우기 추가
- OCR 전에 이미지 대비를 보정하는 전처리 추가
- `촬영 가이드` 카드로 OCR 상태를 쉬운 말로 안내하도록 보강
- 화면 표시 서비스명을 `Study Helper for SA` 중심으로 단순화

### 배포 준비 보강

- 정적 배포용 릴리스 폴더 생성 스크립트 추가
- 더블클릭용 릴리스 생성 실행 파일 추가
- 갤럭시 태블릿 실사용 테스트 가이드 문서 추가
- 임시 HTTPS 터널로 실제 태블릿 테스트 주소 생성 성공
- 임시 링크는 Mac 전원이 켜져 있어야 한다는 설명 문서 추가
- Netlify 업로드용 릴리스 zip 파일 준비
- Netlify 무료 플랜 비용/안전 메모 문서 추가
- 실제 사용 기준 팁을 화면과 문서에 추가
- Netlify 배포 후 ChatGPT 로그인 필요 여부를 문서에 명확히 추가
- 화면에서 `기본 사용은 GPT 로그인 불필요` 문구가 바로 보이도록 보강
- API 비용 없이 GPT를 활용하는 현실적 선택지 문서 추가
- ChatGPT 복사와 공유하기 중심으로 보조 흐름 단순화
- Netlify 업로드 성공, 기본 사이트 주소 확인
- 릴리스 생성 시 최신 zip도 자동으로 다시 만들도록 보강
- Netlify 재배포 후 최신 UI 반영 확인 완료
- PC 화면 재촬영 시 OCR이 약해질 수 있다는 테스트 메모 추가
- 실사용 피드백 기준으로 OCR 2차 보정 읽기와 음성 버튼 상태 표시 보강
- 서비스 제목을 `Study Helper for SA` 하나로 정리하고 `For Tablet` 을 보조 문구로 축소
- 상단 키워드를 설명형 메타 정보처럼 더 작고 가볍게 조정
- `질문하기` 입력 영역으로 용어를 단순화하고 예시 텍스트와 실제 입력 텍스트 색을 구분
- 힌트 카드와 마지막 단계 카드 안에 `자동 읽기` 버튼을 배치
- 음성 입력으로 말한 문제 번호가 입력칸에도 자동 반영되도록 보강
- `촬영 가이드` 카드를 제거하고 마지막 단계에서 정답과 풀이를 더 직접적으로 보여주도록 보강

### 검증

- `python3 scripts/verify_static_app.py` 통과
- `python3 -m unittest discover -s tests` 통과
- 로컬 미리보기 주소 `http://127.0.0.1:8010` 확인
